# 🌟 SubWorld Launcher

**SubWorld Launcher** — это лаунчер для подключения к серверу **SubWorld**, разработанный на основе **JavaX** с использованием HTML-интерфейса. Он автоматически устанавливает все необходимые моды, конфигурации и зависимости, обеспечивая быстрый и удобный старт в вашем приключении по уникальному миру Minecraft. 🚀

---

## 🗺️ О сервере SubWorld

**SubWorld** — это кастомный Minecraft-сервер с тщательно проработанной модифицированной сборкой, уникальными механиками и атмосферой, ориентированной на исследование, выживание и креативную игру.

### Особенности игрового процесса:

- 🛠️ **Кастомные моды** — уникальные модификации, делающие игровой процесс глубже, насыщеннее и непредсказуемее.
- 🔨 **Новые рецепты и крафты** — создавайте уникальные предметы и инструменты, которые невозможно найти в ванильной версии игры.
- 🌱 **Неизведанные биомы и структуры** — откройте для себя новые локации, наполненные редкими ресурсами и постройками.
- ⚔️ **Расширенные механики выживания** — адаптируйтесь к новым условиям и испытаниям, которые не встретишь в обычном Minecraft.

## 🚀 Установка и запуск

1. **Скачайте установщик**  
   📦 Загрузите [`SubWorld-Launcher-Installer.exe`](build.gradle) — он автоматически установит необходимые компоненты, включая Java (если не установлена).

2. **Запустите лаунчер**  
   ⚙️ После установки лаунчер сам скачает клиент, моды и настройки. Никаких дополнительных шагов.

3. **Играйте**  
   🎮 Откройте лаунчер, войдите в аккаунт и подключитесь к нашему серверу!

---

## ✨ Возможности лаунчера

- 🌐 **Полная автоматизация**  
  Установка Java, загрузка клиента, обновление сборки и настройка — всё происходит в фоновом режиме.

- 🧩 **Обновления сборки в один клик**  
  Лаунчер проверяет наличие новых версий сборки и применяет обновления автоматически при запуске.

- 🛠️ **Гибкость и простота**  
  Настройки клиента доступны прямо из интерфейса. Вы можете изменить параметры запуска без необходимости вручную редактировать файлы.

- 🖥️ **Современный UI**  
  Интерфейс лаунчера реализован на HTML/CSS через JavaX, что делает его стильным, отзывчивым и интуитивно понятным.
